ANEXO A – ENGENHARIA DE REQUISITOS DE DOCUMENTOS

 

ESPECIFICAÇÃO DOS CASOS DE USO

 

Preâmbulo

 

Os casos de uso aqui descritos auxiliam a gerência de documentos: diplomas, certificados, comprovantes e outros. Todos os documentos emitidos pelo sistema serão “acompanhados”, ou seja, se um histórico é emitido, então o sistema poderá reconhecer a emissão, quando ela ocorreu e quem emitiu, por exemplo. Este recurso permitirá, por exemplo, que documentos possam ser facilmente validados.

 

1 - Caso de Uso DO.1 – Registrar Documento Digital

 ID: DO.1

  

Este caso de uso tem como foco cadastrar um documento digital para consultas.

 

1.1 - Fluxo de Eventos

     

1.1.1       - Fluxo Básico

 

01-Usuário preenche formulário para Cadastrar Documento Digital;

02- Usuário localiza documento a ser inserido no sistema;

03- Usuário seleciona categoria que o documento pertence;

04- Usuário informa Local de Armazenamento de Documento Digital;

05- Usuário requisita a confirmação da operação;

06-Sistema confirma a operação, cadastrando Documento Digital e gerando um Identificador seqüencial.

 

1.1.2       -  Fluxo Alternativo

 

·         No item 05, o sistema exibe mensagem que o documento não foi localizado, sistema solicita nova tentativa.
