Nota

Alguns elementos em arquivos de modelos do Rose estão associados a outros documentos. Para permitir a "instalação" do ambiente de desenvolvimento em outros diretórios e máquinas, é necessário estabelecer um diretório principal onde todos os documentos serão depositados em relação a este diretório. Para tal, siga os seguintes passos no Rose:

  • Abra o Rose, selecione File e então Edit Path Map
  • No campo "Symbol", digite SISPG_HOME (todas as letras maiúsculas)
  • No campo "Actual Path", digite o diretório de referência (principal)
  • Selecione Add e então Close
