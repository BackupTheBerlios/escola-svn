Especificação de Casos de Uso

 

Preâmbulo

Periodicamente ocorrem processos de seleção. Estes processos são acompanhados pelo SISPG conforme os casos de uso contidos neste documento.

 

 

1.                  UC64 Cadastrar processo de seleção 

Todo processo de seleção deverá ser cadastrado no sistema. Cada processo compreende algumas informações importantes, como período da inscrição e data de entrega dos resultados, por exemplo. Este caso de uso permite que um cadastro de processo de seleção seja identificado e alterado, permite a criação de um novo processo.

 

2.                  UC66 Cadastrar candidato 

Este caso de uso permite que uma pessoa física torne-se um(a) candidato(a) após inscrição em determinado processo de seleção. Através deste caso de uso, os atributos de uma inscrição e do candidato também podem ser alterados.

 

 

3.                  UC65 Relatar resultado de seleção 

Após definido o resultado de um processo de seleção, este deverá ser cadastrado. Para tal é suficiente identificar o processo de seleção e, para cada candidato, fornecer o resultado pertinente.

 

 

 

4.                  UC67 Excluir processo de seleção 

Um processo de seleção pode ser excluído. Contudo, algumas dependências, caso existam, tornam esta operação impossibilitada de ocorre. Por exemplo, caso existam candidatos inscritos no respectivo processo. Neste e noutros casos, uma lista de dependências deverá ser exibida, com possibilidade de impressão. Só após removidas todas as dependências, o processo de seleção poderá ser removido.
