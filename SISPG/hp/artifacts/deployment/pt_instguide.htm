Artefato: Guia de Instalação


Artefato:
Guia de Instalação		 O Guia de Instalação, por conter informações privilegiadas, só será possível o acesso por usuários devidamente autorizados (este guia contém informações sobre a instalação do servidor). 

Para o usuário que apenas deseja ter acesso aos serviços, uma vez devidamente autorizado, basta ter acesso à página onde o sistema estará disponível. Nenhuma outra operação será necessária.
