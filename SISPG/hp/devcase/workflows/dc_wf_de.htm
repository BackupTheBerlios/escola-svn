Notes on the Artifacts 

In addition to the artifacts listed, we will develop a basic data migration utility to enable a data conversion from the previous course registration application at Wylie College. This will be developed using Microsoft Visual Basic using ODBC to interface the old Oracle database.  

Note that the project requires two Acceptance Test Sign-offs, one for each release of the software.

The Training Material is developed as a set of HTML pages in a separate Frontpage Web.

Additional Review Procedures

The Review of the Acceptance Test Results requires a meeting conducted by the Test Manager. All the Team Managers will attend this meeting, in addition to the Project Management, Project Owner and key personnel from the user community.  The Acceptance Sign-off document requires the signature of all attendees.
