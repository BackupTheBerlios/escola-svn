Especificação de Casos de Uso

 

Preâmbulo

Este documento apresenta a descrição de vários casos de uso pertinentes ao cadastramento e manutenção de cargos, pessoas físicas e pessoas jurídicas.

 

 

1.                  UC13 Cadastrar pessoa física 

A inscrição em processo de seleção e membros de banca de exame de qualificação, por exemplo, envolvem pessoas físicas. Este caso de uso permite cadastrar pessoas físicas. Após este cadastro, uma determinada pessoa física pode se envolver em atividades da pós-graduação.

 

2.                  UC14 Selecionar pessoa física 

Permite a seleção de uma pessoa física anteriormente cadastrada. A seleção pode ocorrer por um rico conjunto de atributos. Por exemplo, pode-se consultar pelo número do telefone, pelo setor em que mora, pelo primeiro nome, por possuir certa idade e assim por diante.

 

3.                  UC15 Editar pessoa física 

Uma pessoa física está associada à quantidade significativa de informações que variam ao longo do tempo, por exemplo, endereço. Este caso de uso permite manter tais informações atualizadas. Alguns atributos de pessoas físicas só poderão ser alterados por usuário autorizado, por exemplo, CPF e nome. Outros, como o citado endereço, poderão ser mantidos pelo próprio usuário do sistema.

 

4.                  UC16 Excluir pessoa física 

Excepcionalmente, em situação para a qual não se prevê aplicação, no momento, o cadastro de uma pessoa física poderá ser excluído. Este caso de uso permite registrar este desejo, que será avaliado e, posteriormente, conforme a avaliação, a exclusão será efetuada pelo administrador do sistema. Em conseqüência, a remoção não é executada sumariamente.

 

5.                  UC17 Cadastrar pessoa jurídica 

Ao longo do tempo, o cadastro de pessoas jurídicas pode ser necessário. Este caso de uso permite a criação de uma pessoa jurídica.

 

6.                  UC18 Editar pessoa jurídica 

Após cadastrada, uma pessoa jurídica pode ter alguns de seus atributos alterados. Este caso de uso permite manter atualizados os atributos de uma pessoa jurídica.

 

7.                  UC19 Selecionar pessoa jurídica 

De forma análoga ao caso de uso Selecionar pessoa física, o conjunto de atributos de pessoas jurídicas poderão ser utilizados pelos usuários para a identificação de uma pessoa jurídica em particular.

 

8.                  UC20 Excluir pessoa jurídica 

Idem para pessoa física.

 

 

9.                  UC21 Editar cargo 

Atributos de um cargo podem sofrer alterações ao longo do tempo. Este caso de uso representa a interação correspondente.

 

10.             UC22 Selecionar cargo 

Os atributos de um cargo poderão ser utilizados para a identificação de um cargo em particular. Esta seleção é útil a vários outros propósitos.

 

11.             UC23 Cadastrar cargo 

Inicialmente, nenhum cargo estará disponível. Todos eles serão criados/editados à medida em que forem necessários. Este caso de uso captura a necessidade de se criar um cargo. Após criado, o caso de uso editar cargo ressalta operações típicas que deverão ser realizadas pelos usuários.

 

12.             UC24 Editar pessoa-cargo 

Uma pessoa pode ocupar vários cargos e, em alguns momentos, deixar de exercer alguns deles. Este caso de uso descreve a interação necessária para a manutenção deste tipo de informação.

 

13.             UC25 Excluir cargo 

O usuário autorizado poderá requisitar a exclusão de um cargo. Neste caso, a exclusão não poderá ser realizada imediatamente. Em vez desta abordagem, será enviada uma mensagem para o administrador do sistema. Este sim, após analisar as questões envolvidas e, eventualmente entrar em contato com quem requisitou a exclusão, irá remover o cargo em questão.
