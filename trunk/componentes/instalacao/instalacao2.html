SISPG: Guia de Instalação utilizando o instalador ESCOLA 1.0

Conteúdo da página

  • Nessa página se encontram os principais passos para a instalação do ESCOLA de uma maneira mais fácil para o usuário. Será necessário apenas que o usuário realize o download do JDK e do APACHE TOMCAT (pré-requisitos que estão contidos na página anterior.
  • Após o download e configuração desses dois componentes, favor seguir os passos contidos nessa página.

Instalação

    • Faça o download do pacote de instalação: SetupEscola.jar. Armazená-lo em um diretório da sua preferência;
    • Utilizando o terminal do seu sistema operacional (DOS no caso do Windows) vá ao diretório em que em SetupEscola.jar foi armazenado. Nesse diretório favor digitar o seguinte comando: java -jar SetupEscola.jar. Nesse momento, irá aparecer uma tela de instalação do ESCOLA, semelhante à tela mostrada abaixo:


    • A instalação estará completa quando aparecer a seguinte imagem


    • Para a verificação da instalação correta, é necessário que os seguintes diretórios tenham sido criados:


    • Acessar a pasta SCRIPTS.
      1. Caso esteja utilizando o LINUX, digite o seguinte comando: sh ClassPathBanco.sh. Após o dígito desse comando, digitar esse próximo comando para startar o banco: sh scriptBanco.sh
      2. Caso esteja utilizando o WINDOWS, basta dar um duplo click nos seguintes bats:ClassPathBanco.bat e scriptBanco.bat

    • Instalação do Escola no Tomcat

      •
      1. Iniciar o Tomcat;
      2. Abrir o endereco http://localhost:8080/manager/html;

        Observação: O número da porta podera ser diferente caso o servidor Tomcat tenha sido configurado para uma porta diferente.

        Usuario: admin
        Senha: admin

      3. Na tela de gerência do Tomcat, preencher o campo "Select WAR file to upload"; com o endereço do arquivo ..Escola/war/Escola.war
      4. Pressionar o botao Deploy para carregar o contexto no servidor;
      5. Reinicie o TOMCAT;
    • Primeiro Acesso
      1. No próprio servidor acessar o endereço http://localhost:8080/Escola/

          • Cadatre seu usuario;
          • Após o cadastro uma senha temporária será enviada para o seu e-mail

    • Remoção do ESCOLA


      • Para realizar a exclusão do ECOLA é necessário acessar via terminal a pasta Uninstaller. Nela digitar o seguinte comando: java -jar uninstaller.jar. Após total remoção do projeto, aparecerá uma tela indicando que a exclusão do ESCOLA foi concluída com sucesso!
