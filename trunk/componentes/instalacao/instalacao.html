Pré-Requisitos

Para instalar o Escola 1.0 é necessário que estejam instalados os seguintes pacotes:

  1. Java Runtime Environment 6.0 - download no java.sun.com
  2. Apache Tomcat 6.0 - download no tomcat.apache.org
  3. Hibernate ?.?- download no www.hibernate.org
  4. Apache Ant 1.7.0 - download no ant.apache.org
  5. Banco de Dados HSQLDB - download no sourceforge.net

Download

  • Faça o download do pacote de instalação: ESCOLA 1.0.zip

Instalação

  1. Descompactar o Arquivo zip

    2.
    • Executar o comando abaixo em um diretório temporário

      • unzip Escola.zip

    • Diretórios Criados

      • escola-1.0
      escola-1.0/banco
      escola-1.0/scripts
      escola-1.0/war

  2. Criar o Banco de Dados

    3.
    • Entrar no diretório [diretório temporário]/escola-1.0/banco
    • Executar o script ant Build.xml com o comando: “ant Build.xml” ou somente “ant”

  3. Instalação do Escola no Tomcat

    4.
    • Iniciar o Tomcat;
    • Abrir o endereço http://localhost:8080/manager/html;

      Observação: O número da porta poderá ser diferente caso o servidor Tomcat tenha sido configurado para uma porta diferente.

      Usuário: admin
      Senha: admin

    • Na tela de gerência do Tomcat, preencher o campo “Select WAR file to upload” com o endereço do arquivo [diretório temporário]/escola-1.0/war/ Escola.war;
    • Pressionar o botão “Deploy” para carregar o contexto no servidor;
    • Reinicie o TOMCAT;

Primeiro Acesso

  • No próprio servidor acessar o endereço http://localhost:8080/Escola/
