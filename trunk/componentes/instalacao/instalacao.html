Pré-Requisitos

Para instalar o Escola 1.0 será necessário que estejam instalados os seguintes pacotes:

  • Java Runtime Environment 6.0 - download no java.sun.com
  • Apache Tomcat 6.0 - download no tomcat.apache.org
  • Hibernate- download no www.hibernate.org
  • Apache Ant 1.7.0 - download no ant.apache.org
  • Banco de Dados HQLDB - download no sourceforge.net

Instalação

  • Existem duas maneiras de se instalar o ESCOLA. A explicação da instalação da primeira maneira se encontra logo abaixo. É uma opção mais complicada, pois é necessário realizar o donwload de todos os componentes (o projeto em si, o banco de dados, os scripts, etc) e aí partir para a parte de configuração do projeto.
  • Já para a segunda opção foi criado um instalador. Nesse instalador, já esta incluído o projeto, o banco de dados e os scripts necessários para a execução do ESCOLA. Se optar por essa segunda opção, favor clicar AQUI

    • Criar os seguintes diretórios

      •
      1. Faça o download do pacote de instalação: ESCOLA.war e armazená-lo no diretório war do ESCOLA;
      2. Faça o download do BANCO DE DADOS de instalação: HSQL e armazená-lo no diretório BANCO do ESCOLA;
      3. Caso utilize o WINDOWS faça o download do Scripts do Banco de Dados para o Windows e o salve no diretório SCRIPT do ESCOLA; Após ter realizado o download, favor descomprimi-lo na mesma pasta.
      4. Caso utilize o LINUX faça o download do scriptBanco.sh e do Scripts do Banco de Dados para o Linux e os salve no diretório SCRIPT do ESCOLA; Após ter realizado o download, favor descomprimi-lo na mesma pasta.

    • Criação do Banco de Dados

      •
      1. Entrar no diretório ../Escola/script
      2. Caso esteja utilizando o LINUX, digite os seguintes comandos: sh ClassPathBanco.sh e sh scriptBanco.sh sequencialmente;
      3. Caso esteja utilizando o WINDOWS, basta dar um duplo click no ClassPathBanco.bat e no scriptBanco.bat sequencialmente

    • Instalação do Escola no Tomcat

      •
      1. Iniciar o Tomcat;
      2. Abrir o endereco http://localhost:8080/manager/html;

        Observação: O número da porta podera ser diferente caso o servidor Tomcat tenha sido configurado para uma porta diferente.

        Usuario: admin
        Senha: admin

      3. Na tela de gerência do Tomcat, preencher o campo "Select WAR file to upload"; com o endereço do arquivo ..Escola/war/Escola.war
      4. Pressionar o botao Deploy para carregar o contexto no servidor;
      5. Reinicie o TOMCAT;

    • Primeiro Acesso

      1. No próprio servidor acessar o endereço http://localhost:8080/Escola/

          • Cadatre seu usuario;
          • Após o cadastro uma senha temporária será enviada para o seu e-mail
