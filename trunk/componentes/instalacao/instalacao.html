Pre-Requisitos

Para instalar o Escola 1.0 sera necessario que estejam instalados os seguintes pacotes:

  1. Java Runtime Environment 6.0 - download no java.sun.com
  2. Apache Tomcat 6.0 - download no tomcat.apache.org
  3. Hibernate- download no www.hibernate.org
  4. Apache Ant 1.7.0 - download no ant.apache.org
  5. Banco de Dados HQLDB - download no sourceforge.net

Download

  • Faca o download do pacote de instalacao: ESCOLA 1.0.war
  • Faca o download do script de execucao do banco de dados: Build.xml

Instalacao

  1. Criar os seguintes diretorios

    2.
    • Inserir no diretorio BANCO\HSQLDB, o HQLDB que foi baixado
    • Inserir no diretorio BANCO, o banco do ESCOLA que foi baixado
    • NO WAR, inserir o ESCOLA.war

  2. Criar o Banco de Dados

    3.
    • Entrar no diretorio ../Escola/banco via linha de comando;
    • Caso esteja utilizando o LINUX, digite o seguinte comando: export CLASSPATH=$CLASSPATH: Escola/banco/lib/hsqldb.jar
    • Caso esteja utilizando o WINDOWS, digite o seguinte comando: set CLASSPATH=CLASSPATH%..\Escola\banco\HSQLDB\lib\hsqldb.jar;
    • Caso esteja utilizando o WINDOWS, digite o seguinte comando: set CLASSPATH=CLASSPATH%..\Escola\banco\HSQLDB\lib\hsqldb.jar;
    • java org.hsqldb.Server -database.0 escola -dbname.0 escola

  3. Instalacao do Escola no Tomcat

    4.
    • Iniciar o Tomcat;
    • Abrir o endereco http://localhost:8080/manager/html;

      Observacao: O numero da porta podera ser diferente caso o servidor Tomcat tenha sido configurado para uma porta diferente.

      Usuario: admin
      Senha: admin

    • Na tela de gerencia do Tomcat, preencher o campo "Select WAR file to upload"; com o endereco do arquivo ..Escola/war/Escola.war
    • Pressionar o botao Deploy para carregar o contexto no servidor;
    • Reinicie o TOMCAT;

Primeiro Acesso

  • No proprio servidor acessar o endereco http://localhost:8080/Escola/

      • Cadatre seu usuario;
      • Apos o cadastro uma senha temporaria sera enviada para o seu e-mail
