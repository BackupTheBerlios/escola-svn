Pré-Requisitos

Para instalar o Escola 1.0 será necessário que estejam instalados os seguintes pacotes:

  • Java Runtime Environment 6.0 - download no java.sun.com
  • Apache Tomcat 6.0 - download no tomcat.apache.org
  • Hibernate- download no www.hibernate.org
  • Apache Ant 1.7.0 - download no ant.apache.org
  • Banco de Dados HQLDB - download no sourceforge.net

Instalação

  1. Criar os seguintes diretórios

    2.
    • Faça o download do pacote de instalação: ESCOLA 1.0.war e armazená-lo no diretório war do ESCOLA;
    • Faça o download do BANCO DE DADOS de instalação: HSQL e armazená-lo no diretório BANCO do ESCOLA;
    • Caso utilize o WINDOWS faça o download do scriptBanco.bat e armazená-lo no diretório SCRIPT do ESCOLA;
    • Caso utilize o LINUX faça o download do scriptBanco.sh e armazená-lo no diretório SCRIPT do ESCOLA;

  2. Criar o Banco de Dados

    3.
    • Entrar no diretório ../Escola/script
    • Caso esteja utilizando o LINUX, digite o seguinte comando: sh scriptBanco.sh
    • Caso esteja utilizando o WINDOWS, basta dar um duplo click no scriptBanco.bat

  3. Instalação do Escola no Tomcat

    4.
    • Iniciar o Tomcat;
    • Abrir o endereco http://localhost:8080/manager/html;

      Observação: O número da porta podera ser diferente caso o servidor Tomcat tenha sido configurado para uma porta diferente.

      Usuario: admin
      Senha: admin

    • Na tela de gerência do Tomcat, preencher o campo "Select WAR file to upload"; com o endereço do arquivo ..Escola/war/Escola.war
    • Pressionar o botao Deploy para carregar o contexto no servidor;
    • Reinicie o TOMCAT;

Primeiro Acesso

  • No próprio servidor acessar o endereço http://localhost:8080/Escola/

      • Cadatre seu usuario;
      • Após o cadastro uma senha temporária será enviada para o seu e-mail
